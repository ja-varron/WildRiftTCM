## **INT_TC_001:** Scan to Analytics

> **Summary:** This test case verifies the ability of the app to process scanned results and generate analytics.

**Preconditions:** _Exam Results exists_, _Answer Key exists_

Scenario 1

| # | Steps | Expected Behavior |
|-----|-----|-----|
| 1 | Scan student result. | System should process the result and generate analytics. |
| 2 | Process results | System should display the results in a list. |
| 3 | Open analytics | System should display the analytics. |
